# Product Guide

In order to set up the product properly, you will need to follow several steps to ensure everything is running smoothly. you will need to have a basic understanding with using a PC or laptop.

An additional understanding for frameworks, and libraries, and programming laungage will make it even more easier to interact with the actual code.

We will give a quick look about the technologies & tools that are a must and should be downloaded/installed in your device.

Then, we will set up the necessary infrastructure to run the project. This includes installing the required software and dependencies, configuring the local web server....

Once the infrastructure is set up, we will run the server locally and build the Application and Admin Panel to using them.
