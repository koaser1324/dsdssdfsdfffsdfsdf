## Android Studio

<br />
<img alt="Android Studio LOGO" src="/img/android-Studio.svg" width="120" />
<br />
<br />

Android Studio is the official integrated development environment for Google's Android operating system, built on JetBrains' IntelliJ IDEA software and designed specifically for Android development. It is available for download on Windows, macOS and Linux based operating systems.

it's required for running the mobile Application in an android emulator, or/and for editing the source code, you can download it [from here](https://developer.android.com/).

## Google Chrome

<br />
<img alt="Google Chrome LOGO" src="/img/chrome.png" width="100" />
<br />
<br />

Google Chrome is a cross-platform web browser developed by Google. It was first
released in 2008 for Microsoft Windows, built with free software components from
Apple WebKit and Mozilla Firefox. Versions were later released for Linux, macOS,
iOS, and also for Android, where it is the default browser.

it's required if you want to run the Admin Panel on a browser, you can download it [from here](https://www.google.com/chrome/)

## VScode

<br />
<img alt="Visual Studio Code LOGO" src="/img/vscode.png" width="100" />
<br />
<br />

Visual Studio Code (VSCode) is a lightweight, open-source code editor developed by Microsoft. It is a popular choice among developers due to its extensibility and support for a wide range of programming languages and technologies.

VSCode can be downloaded and installed from the [official website](https://code.visualstudio.com/) for Windows, Mac, and Linux. The installation process is straightforward and should take only a few minutes to complete.

:::note
You can use other Editors like Android Studio, but it is recommended to use Visual Studio Code as the code editor since it's easy to use and very helpful.
:::

## Appainter

<br />
<img alt="appainter LOGO" src="/img/appainter.png" width="200" />
<br />
<br />

A material theme editor and generator for Flutter to configure and preview the overall visual theme of your material app.

This is a tool website, that will make editing the theme, colors, fonts, components styles... very easy for you in order to customize the look of the application, this don't require downloading it, you should only follow those steps [from here](https://appainter.dev).
